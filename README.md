# Computer Science Portfolio

A professional portfolio website showcasing AP Computer Science projects, skills, and growth.

## Overview

This portfolio is built with HTML, CSS, and JavaScript, hosted on GitHub Pages. It showcases:

- **Home Page**: Introduction and welcome
- **About Me Page**: Interests, hobbies, and technology stack
- **Projects Page**: 4+ programming projects with descriptions and lessons learned
- **Experience Page**: Education, skills, activities, and achievements
- **Reflection Page**: 300+ word reflection on growth and learning

## Features

✓ Responsive design that works on all devices
✓ Professional, clean user interface
✓ Easy navigation between sections
✓ Showcases real programming work
✓ Demonstrates version control and GitHub proficiency

## Project Structure

```
.
├── index.html        # Home page
├── about.html        # About Me page
├── projects.html     # Projects showcase

└── README.md         # This file
```

## Pages

### Home (index.html)
Introduction, welcome message, and call-to-action buttons to explore the portfolio.

### About Me (about.html)
Personal interests, hobbies, technology interests, and technical skills.

### Projects (projects.html)
Showcase of 4 programming projects with:
- Project descriptions
- Technologies used
- Key features
- Challenges faced and lessons learned

### Experience (experience.html)
- Education background
- Technical skills by category
- Activities and leadership
- Volunteer work
- Achievements and awards

### Reflection (reflection.html)
In-depth reflection on:
- Journey in computer science
- Challenges overcome
- Personal growth and development
- Future goals and aspirations

## Customization

To customize this portfolio:

1. **Update personal information**: Edit the content in each HTML file
2. **Change colors**: Modify the CSS variables in `styles.css`
3. **Add projects**: Add new project cards to `projects.html`
4. **Update experience**: Modify skills, education, and achievements in `experience.html`
5. **Personalize reflection**: Write your own reflection in `reflection.html`

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (for interactivity)
- GitHub Pages (hosting)
- Git (version control)

## How to Deploy

1. Ensure this repository is named `username.github.io`
2. Push all files to the main branch
3. Enable GitHub Pages in repository settings
4. Access your portfolio at `https://username.github.io`
